# CodSoft Web Development Internship - Task 1

## Personal Portfolio Website

This project is a responsive personal portfolio website developed as Task 1 of the CodSoft Web Development Internship.

The portfolio website was designed to showcase my profile, technical skills, projects, education, resume, and contact information through a clean, modern, and user-friendly interface.

## Features

- Responsive design for desktop, tablet, and mobile devices
- About Me section introducing my background and interests
- Skills section highlighting technical competencies
- Projects section showcasing personal projects
- Resume download functionality
- Contact section with professional information
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3

## Learning Outcomes

Through this project, I improved my understanding of:

- Semantic HTML structure
- CSS styling and layouts
- Responsive web design
- Flexbox and Grid concepts
- User interface design principles
- Building professional portfolio websites
